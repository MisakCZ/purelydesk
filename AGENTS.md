# Project rules

- Framework: Laravel
- Database: MariaDB
- Auth will later integrate with KeyShield SSO
- UI: Blade templates, no SPA
- Keep code simple and conventional
- Do not add new packages unless explicitly requested
- Public and internal ticket comments must stay separate
- Ticket visibility must support public and restricted modes
- Prepare clean migrations, models, controllers, and policies
- Do not modify deployment/server config unless requested
